# Braille Signage

> Improving accessibility at Daegu International School through 3D-printed Braille signage.

## Overview

Braille Signage is a collaborative project between the **FAB LAB** and the **Disability Awareness Club** at **Daegu International School (DIS)**. This initiative was created to improve accessibility for individuals with visual impairments by designing and manufacturing durable, 3D-printed Braille signage for use around the school campus.

While DIS provides a supportive learning environment, we identified a lack of tactile navigation and accessibility resources for visually impaired students, staff, and visitors. This project aims to address that gap through an open-source, scalable signage system that can be continuously improved and expanded.

This repository contains the design files, documentation, and development history of our Braille signage prototypes.

## Prototype 1

The first prototype focused on establishing a functional proof of concept.

### Features

- Raised Braille dots
- Printed text labels
- Basic rectangular design
- Initial evaluation of Braille sizing and spacing
- Test prints for manufacturing consistency

### Lessons Learned

- Braille spacing required refinement.
- Improved mounting methods were needed.
- Better tactile readability was achieved through adjustments to dot height and spacing.

---

## Prototype 2

The second prototype incorporates feedback from the initial design and introduces several improvements.

### Improvements

- Refined Braille spacing
- Improved tactile readability
- Cleaner typography
- Stronger structural design
- Better mounting solution
- Higher-quality surface finish
- Easier installation

Prototype 2 represents a more practical design intended for future deployment throughout the school.

## Manufacturing

The signage is produced using additive manufacturing (3D printing), providing several advantages:

- Rapid prototyping
- Low production cost
- Easy customization
- Fast replacement of damaged signage
- Scalable production for additional school locations
